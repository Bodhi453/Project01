# Project 01 – Workout Website

## Overview
This project is a multi-page website about a workout program created by Jeff Nippard.  
The goal of this project is to practice building a website using HTML and CSS.

The website includes:
- A workout plan page
- A technique page
- An about page

---

## Pages

### Home (Workout Plan)
This page shows the full workout program using tables.  
It includes exercises, sets, reps, and rest times.

### Technique Page
This page explains how to properly perform exercises and gives helpful tips.

### About Page
This page gives background information about Jeff Nippard and where the program comes from.

---

## Features

- Uses multiple linked pages
- Includes tables for workout data
- Uses an image
- Styled with CSS
- Navigation bar on every page
- Basic responsive design (nav bar adjusts on smaller screens)
